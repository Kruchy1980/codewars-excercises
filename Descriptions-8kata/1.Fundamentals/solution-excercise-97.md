# You Can't Code Under Pressure #1

Code as fast as you can! You need to double the integer and return it.

### Solution

function doubleInteger(i) {

##### It is enough to use assignment operator to make the number doubled

    i \*= 2;  <!-- = i = i \* 2-->

    return i;

}
